How  to  bind  events?

  • Inline

    
<input type="submit" name="submit" value="Submit" onSubmit="validate_form()">

  • Specific

    
$('input[type="submit"]').submit(function(event){
    .......
});

  • Delegate

    
$('form').on('submit', 'input[type="submit"]', function(event){
    .....
});

How  to  unbind  events?

  • Unbind All

    
$('form').off();

  • Unbind Specific Events

    
$('form').off('submit');

jQuery.ajax();

Commonly  Used  Options


jQuery.ajax({
    url         : URL,

    type        : "Default: GET|POST|PUT|DELETE",

    dataType    : "xml|html|script|json",

    contentType : "Default: application/x-www-form-urlencoded|text/json|text/xml",

    success     : function(data){
        console.log("Successful Request");
    },

    error       : function(data){
        console.log("Oops!! Something went wrong");
    },

    complete    : function(data){
        console.log("Executed when the call is completed");
    }
});
